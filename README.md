### ✨ 这是什么？
**豆包喵喵** 是一款极简、可爱且强大的跨设备输入同步工具。
想象一下：你躺在沙发上，拿着手机，对着手机说出一段话（或者用手机飞快地打字），而你电脑桌上的光标，就像被魔法驱动一样，**实时、流畅**地敲出了同样的内容。

它专为**语音输入重度用户**和**多设备协同**场景打造，解决了“手机打字快，电脑编辑爽”的终极痛点。

### 🚀 核心亮点

#### 1. ⚡ 毫秒级实时同步
*   **零延迟感**：通过 WebSocket 局域网传输，手机敲下一个字，电脑屏幕上几乎同时出现。
*   **全平台兼容**：无论你用的是 Windows 还是 Mac，无论手机是 iOS 还是安卓，只要在同一个 Wi-Fi 下，无需安装手机 App，打开网页即用。

#### 2. 🧠 独创“电脑介入”智能模式 (Smart Rebase)
这是豆包喵喵最懂你的地方。它完美解决了传统同步工具“打架”的问题：
*   **无缝切换**：当你觉得手机输入的一段话不够好，直接伸手在电脑键盘上修改，或者点击鼠标移动光标。
*   **智能感知**：喵喵会立刻感知到你在使用电脑，并自动**“切断”**当前的上下文。
*   **互不干扰**：当你回到手机继续输入时，它会把新内容作为一段**全新的文字**发送，绝对**不会**倒回去删改你刚才在电脑上辛苦修改的内容。
*   **标点自动优化**：智能去除新段落开头的逗号、顿号，让续写如丝般顺滑。

#### 3. 😻 治愈系撸猫体验
这不仅仅是一个工具，更是一个电子宠物：
*   **交互式猫咪**：屏幕上趴着一只可爱的小猫。当你打字时，它的两只小爪子会飞快地敲击键盘，甚至还能看到它按键的微动作！
*   **萌系 UI**：采用了柔和温暖的莫兰迪配色和圆润的“站酷快乐体”，让你在码字时心情愉悦。
*   **微动效细节**：眨眼、悬浮气泡、动态阴影……每一个细节都充满了生命力。

#### 4. 🛠️ 极客级可玩性
*   **单文件运行**：虽然功能强大，但核心代码仅一个 `server.py` 文件，绿色免安装。
*   **便携模式**：提供 Portable 版打包脚本，生成的 `.exe` 只有几 MB，放在 U 盘里随插随用。
*   **高度可配置**：发送延迟、自动清空时间、键盘检测开关，想怎么调就怎么调。

### 📖 如何使用？

1.  **电脑端启动**：双击 `豆包喵喵.exe`（或运行源码）。
2.  **手机端连接**：
    *   电脑屏幕黑框里会显示一个地址，例如 `http://192.168.1.5:5000`。
    *   手机打开浏览器，输入这个地址。
3.  **开始魔法**：
    *   在手机屏幕上的“横线纸”区域，呼出你的输入法（推荐使用**豆包**、**讯飞**或**微信键盘**的语音输入）。
    *   看着电脑屏幕，见证文字飞舞的奇迹！
